# Go Set

Go Set Package is a simple library for set operations with generic type supported.


## Installation

```
	go get "github.com/athom/goset"
```

## Useage

#### Detections

###### 1. IsUniq
###### 2. IsEqual
###### 3. IsIncluded
###### 4. IsSubSet
###### 5. IsSuperSet


#### Operations
###### 1. Uniq
###### 2. Intersect 
###### 3. Union
###### 4. Difference
###### 5. AddElement
###### 6. AddElements
###### 7. RemoveElement
###### 8. RemoveElements
